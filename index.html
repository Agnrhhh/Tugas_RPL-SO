TUGAS SISTEM OPERASI

Materi 1

    Diketahui :

    Jumlah track = 2000,

    Posisi head berada di track : 1234

    Urutan track yang perlu diakses yaitu :

    1600, 1050, 1775, 1950, [x1], 825, 75, 1300, 1450, 600, [x2], 1600, 1050, 30, 1900, [x3], 700, 1300, 55, 425, [x4], 700, 1600, 200.

    Lakukan perhitungan banyaknya proses lintas track yang dilakukan menggunakan ketujuh algoritma (mulai dari PTPD hingga Circular Scan), dan gambarkan grafik perjalanan lintasan yang dilakukan serta tambahkan sedikit narasi sebagai penjelasannya.

Catatan:

    [x1] adalah tanggal lahir masing-masing x 10, misal : 1 menjadi 10, 29 menjadi 290

    [x2] adalah bulan lahir masing-masing x 100, misal : 4 menjadi 400, 11 menjadi 1100

    [x3] adalah 4 digit tahun lahir masing-masing, misal : 2007, 2010

    [x4] adalah 3 digit terakhir dari NIM, misal 395, 056 menjadi 56



Jawaban

    Tanggal lahir : 03-06-2002

    NIM : 2303010267

    [x1] (Tanggal lahir x 10): 3 x 10 = 30

    [x2] (Bulan lahir x 100): 6 x 100 = 600

    [x3] (Tahun lahir): 2002

    [x4] (3 Digit Terakhir NIM): 267



1. FCFS (First Come First Serve)

Dari 1234 ke 1600 → |1600 - 1234| = 366

1600 ke 1050 → |1050 - 1600| = 550

1050 ke 1775 → |1775 - 1050| = 725

1775 ke 1950 → |1950 - 1775| = 175

1950 ke 30 → |30 - 1950| = 1920

30 ke 825 → |825 - 30| = 795

825 ke 75 → |75 - 825| = 750

75 ke 1300 → |1300 - 75| = 1225

1300 ke 1450 → |1450 - 1300| = 150

1450 ke 600 → |600 - 1450| = 850

600 ke 600 → |600 - 600| = 0

600 ke 1600 → |1600 - 600| = 1000

1600 ke 1050 → |1050 - 1600| = 550

1050 ke 30 → |30 - 1050| = 1020

30 ke 1900 → |1900 - 30| = 1870

1900 ke 2002 → |2002 - 1900| = 102

2002 ke 700 → |700 - 2002| = 1302

700 ke 1300 → |1300 - 700| = 600

1300 ke 55 → |55 - 1300| = 1245

55 ke 425 → |425 - 55| = 370

425 ke 267 → |267 - 425| = 158

267 ke 700 → |700 - 267| = 433

700 ke 1600 → |1600 - 700| = 900

1600 ke 200 → |200 - 1600| = 1400

Total lintasan FCFS = 366 + 550 + 725 + 175 + 1920 + 795 + 750 + 1225 + 150 + 850 + 0 + 1000 + 550 + 1020 + 1870 + 102 + 1302 + 600 + 1245 + 370 + 158 + 433 + 900 + 1400

= 19,191 tracks
